# af-test-developer

graph-autofusion 项目 UT/ST 测试开发辅助 Skill。

## 功能

- 辅助编写 super_kernel/autofuse 的单元测试和系统测试
- 指导测试编译和运行流程
- C++ 测试规范（gtest + mockcpp）
- Python 测试规范（pytest）
- 覆盖率报告生成

## 触发场景

用户提到以下关键词时自动触发：
- 写测试、UT、ST、单元测试、系统测试
- gtest、mockcpp、pytest
- 测试覆盖率、coverage
- 运行测试、测试编译

## 验证方法

在 opencode 中输入以下指令验证 Skill 是否生效：
- "如何运行 SuperKernel 的 UT？"
- "帮我写一个 gtest 测试用例"
- "如何生成覆盖率报告？"
